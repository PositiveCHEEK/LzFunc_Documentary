LzFunc - Less Boilerplate

This is a documentation to a Python module called "LzFunc". The module includes some functions that can simplify the code and reduce the "boilerplate", so the programmer doesn't have to grind down his fingers to write 5 more characters.

There is a full documentation for all the functions LzFunc that includes:

Downloading LzFunc

The process is simple. You just have to enter this line in the console with "pip" installed:

pip install LzFunc

Then you can open your .py file and import all of the functions from the module by your preferred choice, e.g.:

from LzFunc import *

By doing these steps, you'll be able to use all LzFunctions in your algorithm. You can also check out a built-in documentation by typing this line:

help(core)
nl()

Skips a line in the output. The argument determines how many lines should be skipped.

print("Hello")
nl(2)
print("World")
> Hello
>
>
> World
pause()

Waits an interval with a three-dot animation. The argument determines how many seconds should be awaited before continuing. Might be used as a fake loading or as a visual effect.

print("Hello")
pause(3)
print("World")
> Hello
input_yesno()

Makes simple "Yes or No" question that returns boolean depending on user's answer. The string argument determines question asked to the user. (Has pre-defined yes or no answers: yes, y, ofc, of course, duh, yeah, yea, no, n, nah, naw, nuh uh).

answer = input_yesno("Do you like dancing? ")
if answer == True:
    print("I like dancing too!")
else:
    print("Too bad.")
> Do you like dancing? Yes
> I like dancing too!
input_default()

Makes simple question that returns either user value or default value if the first one was empty string. First string argument determines question asked to the user. The second one determines default answer.

name = input_default("What is your name? (Anonym by Default) ", "Anonym")
print(name)
> What is your name? (Anonym by Default) [ENTER]
> Anonym
invert()

Simple switch to invert any variable. Argument is the variable that supports inversion. The supported variable types are as follows: bool, int, str, list, None and "All".
* No way it's simple, I spent half an hour to make this function.

number = int(input("Enter number: "))
turn = false

if number > 0:
    turn = input_yesno("Invert the number? ")
if turn:
    number = invert(number)
print("Current number:", number)
> Enter number: 6
> Invert number? Yes
> Current number: -6

dictind()
dictind() is deprecated and shouldn't be used. It will be deleted in the next update.

Returns value from a dictionary. Supports keys, values, items.

shoplist = {"fruits":"apple","vegetables":["carrot","potato"]}
print(dictind(shoplist, 2, "value", 1))
> potato
isit()

Checks numbers based on the selected mode. The first argument is a variable, supported types depend on mode. The second argument is mode, represented as string and supports "Natural", "Whole" and "Palindrome" modes. Integer argument is supported for all modes, string is supported only for "Palindrome" mode.

pal = 1225221
print(isit(pal, "palindrome"))
> True
listop()

Does an operation with two lists. The first and second arguments are lists. The third one is the operation, represented by string. Supported operations are plus and minus.

pos = [3,4]
dpos = [-1,1]
pos = listop(dpos,pos,"+")
print(pos)
> [2, 5]

0.1.2

What is this?
This is a LzFunc's Extra class for the invert() function! It was meant to be the opposite side for the None, because if the opposite would be anything else, it would look like a bit off.

The All itself isn't very functional, it will raise "NonSpecificError" if you'll try to call it, but if you SPECIFY it, you can actually... use it? Not that it has something special, if you try to call it using All("a"), it will be just "a" and nothing else.
